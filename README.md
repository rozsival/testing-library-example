# React Testing Library Example

ApiTree Academy example of React Testing Library for components and hooks.

## 💾 Installation

```bash
nvm use
yarn install
```

## 👨‍💻 Development

```bash
yarn start
```

## 🧪 Testing

```bash
yarn test
```
